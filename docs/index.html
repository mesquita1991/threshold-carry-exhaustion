Main theorem

The result in one statement

For threshold carry Hoeffding matrices $B_{q,r,t}$, with every base $q\ge 2$, every integer threshold $t$, and every number of summands $4\le r\le 32$, the singular parameters are exactly the union of:

$$\text{borders} \;\cup\; \text{carry bands} \;\cup\; \text{self-dual thresholds}. $$

This is a certified exhaustion theorem: no additional singularities occur in that range.

Pedagogical proof architecture

How the argument is organized

The site separates the mathematical core from the computational certification. The argument is easier to audit when read as a pipeline.

1

Start with $B_{q,r,t}$

The original object is the carry-threshold Hoeffding matrix. Singularities are the main target.

2

Reduce to a residual matrix $R$

Known singular families are peeled away, leaving a residual problem that is algebraically sharper.

3

Move to a coefficient map $A$

Confluent interpolation and Fourier-based changes of basis convert the residual matrix into a coefficient model.

4

Isolate $D_{n,p,a_0}(q)$

The remaining obstruction is encoded by an explicit Toeplitz–Padé-type determinant.

5

Certify stable families

For the stable region, one proves there are no integer roots in the relevant range or factors the symbolic exceptions exactly.

6

Audit unstable families directly

Remaining parameters are checked by exact arithmetic in an independent verification path.

Software role in the proof

The code is not ancillary decoration. It freezes the exact certificate families and the exact unstable-case audit used by the theorem. The site therefore treats software verification as part of the mathematical evidence, not as a side note.

This presentation follows the practical view that computational proofs should expose assumptions, exact artifacts, and verification pathways explicitly.

Limits and frontier

What this work does not claim

It does not prove exhaustion for all r.

The theorem is restricted to $4\le r\le 32$.

It does not solve the deep self-dual corank problem uniformly.

The self-dual structure is isolated, but the fully general deep regime remains open.

It does not provide a complete global inertia classification.

Several spectral and inertia questions remain open beyond the certified exhaustion result.

It does not use the later candidate matrix K as part of the main theorem.

The published principal article intentionally excludes that later exploratory line.
