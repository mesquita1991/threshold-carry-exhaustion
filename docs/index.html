Main theorem

Exhaustion for 4 ≤ r ≤ 32

For every base q ≥ 2 and every integer threshold t, singularity occurs exactly in the known structural families:

singular ⇔ boundary ∪ carry bands ∪ self-dual

No floating-point arithmetic is used in the certification.

The mathematical statement

One precise result, with explicit scope

The work classifies singularity for a finite but nontrivial range of summand counts, uniformly in the base and threshold.

Object of study

Let Bq,r,t be the reduced matrix of the second-order Hoeffding interaction associated with the event that the sum of r independent uniform base-q digits is at least t.

The classification problem asks for all parameters for which this structured matrix is singular.

Certified conclusion

For every q ≥ 2, every integer t, and every 4 ≤ r ≤ 32, the known boundary, carry-band, and self-dual families exhaust all singularities.

The statement is exact and computer-assisted, not numerical.

General mechanism

For each fixed residual degree, the infinite problem in the base q reduces to finitely many exact integer computations.

Scientific limits

What this release does not claim

The boundaries below are part of the theorem statement, not caveats hidden in supplementary material.

Not claimed

Arbitrary numbers of summands

The exhaustion theorem is certified for 4 ≤ r ≤ 32, not for all r.

Not claimed

Uniform nonvanishing in every residual degree

Finite decidability is proved degree by degree; a universal product formula remains open.

Not claimed

Deep self-dual corank classification

The unresolved deep self-dual rank problem is deliberately excluded from this paper.

Not claimed

Parameter-only global inertia formula

The release concerns singularity exhaustion, not a complete regional inertia classification.
